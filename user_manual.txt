To access the web-app:
Type the IP which the web-app is hosted on. If you are not logged in, it will automatically redirect you to the login page. 

If you are logged in, it will automatically redirect to http://{hosted-ip}/dashboard/

After selecting the client code, select factory code from the Factory Code dropdown.
After selecting factory code from the dropdown, select line number from the Line Number dropdown.
